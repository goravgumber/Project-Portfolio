Personal portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

Minimalist, dark-themed, engineering-focused. Designed to reflect a backend/systems engineering mindset — clean architecture, modular components, and performance-first delivery.

## Sections

- **Hero** — Terminal-styled intro with location, role, and CTA links
- **About** — Background, interests, and engineering focus areas
- **Tech Stack** — Categorized skills: Languages, Backend & APIs, Databases & Infrastructure, AI & Systems, Developer Tools
- **Experience** — Software Developer Intern at Infotact Solutions
- **Featured Projects** — 7 projects with architecture descriptions, tech tags, GitHub, and live links
- **Engineering Philosophy** — 6 principles that guide how I build systems
- **Open Source** — GitHub presence, 100DaysOfReact, learning roadmap
- **Contact** — Email, GitHub, LinkedIn, location

## Tech Stack

| | |
|---|---|
| Framework | Next.js (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| Animation | Framer Motion |
| Icons | Lucide React + Custom SVG |
| UI | shadcn/ui primitives |
| Deployment | Vercel |

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
src/
├── app/              # Pages and layout
├── components/
│   ├── ui/           # Reusable primitives
│   └── sections/     # Page sections
├── data/             # Content (edit to customize)
└── lib/              # Utilities
```

## Customize

Edit files in `src/data/` to update:

- `info.ts` — Personal info, about text, engineering philosophy
- `tech-stack.ts` — Skills and categories
- `experience.ts` — Work experience
- `projects.ts` — Project details

Replace `public/resume.pdf` with your resume.
